Lab Focus: Fundamental paradigms, data concepts, and algorithm mechanics.

1. Core Concepts

Machine Learning (ML) is a subset of AI that focuses on building systems that learn from data to improve performance without being explicitly programmed.

The Model

A mathematical representation trained to recognize patterns. It maps Features (inputs) to Targets (outputs).

Training vs. Test

Training Data: Used to teach the model.
Test Data: A separate set used to evaluate real-world performance.

1.1 Paradigms

  • Supervised Learning: Learning from labeled data (Input + Correct Answer).
  • Unsupervised Learning: Discovering hidden structures in unlabeled data.
  • Reinforcement Learning: Learning through trial and error via rewards and penalties.

2. Model Performance & Trade-offs

The goal of ML is to find the "Sweet Spot" between simplicity and complexity.

Bias (Underfitting)

The model is too simple to capture the relationship. High error on both training and test data.

Variance (Overfitting)

The model is too complex and learns random noise. High training accuracy, but fails on new data.

3. Supervised Learning Algorithms

Algorithm Type Logic
Linear Regression Regression Fits a straight line to predict numeric values.
Logistic Regression Classification Uses a Sigmoid function to predict probabilities.
K-Nearest Neighbors Both Predicts based on the "K" most similar examples.
SVM Classification Finds the "Hyperplane" with the widest margin.
Decision Trees Both A series of binary (Yes/No) questions.
Naive Bayes Classification Probabilistic classifier based on Bayes' Theorem.

4. Unsupervised & Deep Learning

K-Means Clustering

The primary tool for grouping unlabeled data. It iteratively moves Centroids until it finds stable clusters.

Deep Learning (Neural Networks)

Mimics human brain structure with Hidden Layers. It is capable of automatically extracting complex features from raw data like pixels or audio.

5. Optimization Terms

• Epoch: One full pass through the dataset.

• Learning Rate: The step size taken towards the minimum loss.

• Hyperparameter: Settings defined before training (e.g., K in KNN).

• Loss Function: A mathematical measure of the model's error.
