Basic Concepts and Definitions

State Space

  • State: A representation (formulation) of the task in the process of its solution.
    • Initial State Represented with S
    • Intermediate State Any capital letter except S and G
    • Goal State Represented with G (G1, G2, etc.)
  • Successor Function: (Operator) Obtaining one state from another.
  • Path Cost: Additive; e.g., sum of distances, number of actions executed, etc.
  • State Space: The totality of all possible states that can be obtained from a given initial state.
"A solution is a sequence of actions leading from the initial state to a goal state."

Representation of the State Space

The representation of the state space is a graph or a tree, where each state is a node and the successor function is an edge.

When represented as a tree, it is a Search Tree:

  • The initial state is the root.
  • Terminal and goal states are represented as leaves.
bMaximum branching factor of the search tree
dDepth of the least-cost solution
mMaximum depth of the state space (may be ∞)

Global Search vs. Local Search

Global Search: Look all over the state space. All states may be traversed.

Local Search: Look only at the local area. Used as an optimizing algorithm to find solutions quickly when the path is not the priority.

  • Maximizes a specific criterion among candidate solutions.
  • Applies local changes until an optimal solution is found.

Uninformed (Blind) Search

Uninformed strategies use only the information available in the problem definition. They have no knowledge of where the goal might be.

Strategies:

Depth-First Search (DFS) Breadth-First Search (BFS) Uniform-Cost Search (UCS) Depth-Limited Search (DLS) Iterative Deepening Search (IDS)
