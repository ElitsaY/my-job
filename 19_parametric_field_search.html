Overview

Parametric search and field search are used in structured search systems like e-commerce, digital libraries, and databases.

1. Field Search

Definition

Field search restricts query terms to specific document fields such as title, author, abstract, price, or date.

Example

Document fields
Title: Introduction to IR
Author: Manning
Year: 2008
Publisher: Cambridge
Query
title:"information retrieval"
author:Manning

This means: search the phrase only in the title field and match “Manning” only in the author field.

Example Systems

  • Google (site:, filetype:)
  • Amazon (brand:, category:)
  • PubMed (author:, journal:)

How It Works Internally

Instead of one inverted index, maintain separate postings lists per field.

Field-specific postings
title:cat  → [Doc1, Doc3]
body:cat   → [Doc1, Doc2, Doc3]
author:cat → []

Why It’s Useful

  • More precise search.
  • Reduces irrelevant results.
  • Supports structured filtering.

2. Parametric Search

Definition

Parametric search restricts results using structured attributes such as numeric ranges, dates, categories, or boolean flags.

Example (E-commerce)

Query
laptop
brand = Dell
price between 500 and 1000
rating ≥ 4

Example (Travel)

Query
Flights NY → LA
Date: March 10
Price < $300
Non-stop only

How It Works Internally

Numeric and date fields use structured indexes such as B-trees, range trees, or sorted arrays.

Price index (example)
500  → Doc1
700  → Doc2
1200 → Doc3

Range query price 500–1000 returns Doc1 and Doc2.

4. Combined Search (Most Common)

Example
title:"machine learning"
author:Smith
year > 2015

Process:

  • Text conditions → inverted index.
  • Numeric filters → range index.
  • Intersect results.

5. Query Processing Example

Doc Title Year
D1Machine Learning2010
D2Deep Learning2018
D3Machine Vision2020
Query
title:machine
year > 2015

Step 1: title index → {D1, D3}

Step 2: year filter → {D2, D3}

Step 3: intersection → {D3}

Result: D3

6. Why These Are Important

  • Users want precise filtering.
  • Pure keyword search is not enough.
  • Structured data improves relevance.

7. Where You See This Daily

  • Amazon filters (price, brand, rating).
  • Google advanced search.
  • LinkedIn filters (location, job title, experience).

8. Intuition

Field search answers: “Search this word in this specific part of the document.”

Parametric search answers: “Filter documents that satisfy these structured conditions.”

9. Exam-Ready Definitions

Field Search: A search technique that restricts query terms to specific document fields such as title, author, or abstract.

Parametric Search: A search method that applies structured constraints (e.g., numeric ranges, dates, categories) to filter documents based on metadata attributes.
